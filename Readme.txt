How to run the project:
	Server:
		-deploy oracle database on your machine.
		-create new database with name "chat" and password "chat".
		-double click on the server jar.
		-open cmd and get your ip by "ipconfig" command.
		-put the ip address.
		-double click on start button to start server.
	Client:
		-double click on the client jar.
		-enter server ip address.
		-signUp or login to start chat.
